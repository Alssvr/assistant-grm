# Ассистент по нормализации данных ГРМ

Локальный AI-ассистент для поиска информации в методиках нормализации данных.

## Установка

1. Установите Ollama: https://ollama.com/
2. Скачайте модель: `ollama pull gemma3:4b`
3. Установите зависимости: `pip install -r requirements.txt`

## Запуск

```bash
python run.py
